# partial quantile score candidates constructor

See documentation for
[`make_quantile_score_candidates()`](https://docs.opendp.org/reference/make_quantile_score_candidates.md)
for details.

## Usage

``` r
then_quantile_score_candidates(lhs, candidates, alpha)
```

## Arguments

- lhs:

  The prior transformation or metric space.

- candidates:

  Potential quantiles to score

- alpha:

  a value in \\(\[0, 1\]\\). Choose 0.5 for median

## Value

Transformation

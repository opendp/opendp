# partial population amplification constructor

See documentation for
[`make_population_amplification()`](https://docs.opendp.org/reference/make_population_amplification.md)
for details.

## Usage

``` r
then_population_amplification(lhs, measurement, population_size)
```

## Arguments

- lhs:

  The prior transformation or metric space.

- measurement:

  the computation to amplify

- population_size:

  the size of the population from which the input dataset is a simple
  sample

## Value

Measurement

# partial chain pm constructor

See documentation for
[`make_chain_pm()`](https://docs.opendp.org/reference/make_chain_pm.md)
for details.

## Usage

``` r
then_chain_pm(lhs, postprocess1, measurement0)
```

## Arguments

- lhs:

  The prior transformation or metric space.

- postprocess1:

  outer postprocessor

- measurement0:

  inner measurement/mechanism

## Value

Measurement

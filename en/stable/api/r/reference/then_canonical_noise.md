# partial canonical noise constructor

See documentation for
[`make_canonical_noise()`](https://docs.opendp.org/reference/make_canonical_noise.md)
for details.

## Usage

``` r
then_canonical_noise(lhs, d_in, d_out)
```

## Arguments

- lhs:

  The prior transformation or metric space.

- d_in:

  Sensitivity

- d_out:

  Privacy parameters (ε, δ)

## Value

Measurement

# approximate constructor

Constructs a new output measurement where the output measure is
δ-approximate, where δ=0.

## Usage

``` r
make_approximate(measurement)
```

## Arguments

- measurement:

  a measurement with a privacy measure to be casted

## Value

Measurement

Required features: `contrib`

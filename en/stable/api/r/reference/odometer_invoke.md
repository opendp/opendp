# Invoke the `odometer` with `arg`. Returns a differentially private release.

Invoke the `odometer` with `arg`. Returns a differentially private
release.

## Usage

``` r
odometer_invoke(this, arg)
```

## Arguments

- this:

  Odometer to invoke.

- arg:

  Input data to supply to the odometer. A member of the odometer's input
  domain.

# partial cdf constructor

See documentation for
[`make_cdf()`](https://docs.opendp.org/reference/make_cdf.md) for
details.

## Usage

``` r
then_cdf(lhs, .TA = "float")
```

## Arguments

- lhs:

  The prior transformation or metric space.

- .TA:

  Atomic Type. One of `f32` or `f64`

## Value

Function
